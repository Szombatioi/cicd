# Felhők hálózati szolgáltatásai laboratórium - CI/CD 1. labor

A repository a `Felhők hálózati szolgáltatásai laboratórium` tantárgy első és második CI/CD laborjának a munkakörnyezete.

A laborok célja megismertetni a hallgatókat a *Continuous Integration / Continuous Delivery/Deployment* folyamataival.
